# MSSC Beer Order Service

This repository demonstrates partial decomposition of a monolith application into a Domain-Driven Design architecture. This follows [John Thompson (aka springframeworkguru)](https://github.com/springframeworkguru) and his corresponding Spring Framework Udemy course.

This repository implements the Beer Order Service wherein the following operations are managed:
- Beer orders from placement to shipment
- Customers
- Call backs to customers
